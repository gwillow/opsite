# 综合运维管理系统

​        采用Django框架开发的一个运维管理系统，可方便实现设备的录入修改等管理及各类任务管理等运维操作。

后台结合Ansible实现通过界面对远程主机进行常规管理，所需要python模块请参见requirements。
